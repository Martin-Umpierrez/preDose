preDose

preDose: An R-package for Robust External Evaluation of popPKPD Models.

preDose is a free and open source package that automatize the process of external evaluation process using an independent dataset from which the original popPK model was developed.

Currently, the user can choose to use the package based on eitherlixoftconnectors or via mapbayr depending on whether lixoftconnectors(which comes with the installation of Lixoft software) is available in their environment.

You can perform an external evaluation for a single model from :

  • a population PK model (coded in mrgsolve, or from Monolix,
  • a data set with concentrations (NONMEM format or Monolix format)

The package requires the user to provide a popPK model and an external dataset, regardless of the processing engine they choose. Below are the functions established for both utilities.

Installation

preDose is available on CRAN

You can install the development version of preDose from GitHub with:

install.packages("devtools")
devtools::install_github("Martin-Umpierrez/R.preDose")

This is a basic example which shows you how to solve a common problem for external evaluation.
